# Langley

MVP веб-тренажёра по запоминанию иностранных слов.

## Суть приложения

### Настройка

Пользователь выбирает:
- Свой родной язык.
- Изучаемый язык.
- Наборы слов из доступных.

### Рутина

- На экране лента карточек со словами на изучаемом языке.
- Одновременно видно несколько активных карточек.
- Пользователь может переворачивать каждую карточку.
- На оборотной стороне это же слово на родном языке.
- Пользователь может спайпать карточки вправо/влево.
- Свайп влево означает, что слово хорошо знакомо.
- Свайп вправо — что слово ещё не закреплено в памяти.
- Место освобождается — лента двигается (бесконечно).
- Хорошо знакомые слова выпадают реже, незакреплённые — чаще.
- Рутина никогда не заканчивается, только явный выход.

## Ограничения

Приложение frontend-only. Будучи MVP, оно симулирует
запрос доступных наборов слов с сервера, но фактически они "зашиты"
в клиент.

Доступны языки: английский, испанский, русский.
Предусмотрено добавление других языков. Предустановлены наборы слов:
- 100 самых частых существительных;
- 100 самых частых глаголов.

Результатом сборки приложения являются статические файлы:
html, css, js, assets.

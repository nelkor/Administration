# Стек

- TypeScript
- React
- Vite
- CSS Modules
- CSS custom properties для темизации
- Motion
- Vitest

Допускается подключение хорошо документированных библиотек для отдельных задач,
если это не усложняет архитектуру.

## Формат данных

Слова хранятся в массивах объектов, где ключи объектов — коды языков.

Пример:

```js
[
  {
    en: "food",
    ru: "еда",
    es: "comida"
  }
]
```

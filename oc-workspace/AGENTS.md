# AGENTS.md

## Main Agent

- Общается с пользователем.
- Проводит исследования, brainstorming.
- Предлагает варианты решения.
- Управляет ветками репозитория.
- Создаёт и редактирует артефакты.

Имя файла артефакта основного агента начинается с `ma-`.

Основной агент вызывает субагента только с одной конкретной задачей.

## Subagent

- Фокусируется на одной конкретной задаче.
- Не предлагает, а принимает решения.
- Определяет путь развития продукта.
- Определяет структуру и архитектуру продукта.
- Определяет используемые технологии.
- Пишет код в репозитории.
- Создаёт артефакты.

Перед началом любой работы субагент **обязан** прочитать `README.md`
в корне репозитория и все актуальные артефакты.

Результатом любой работы является создание артефакта.
Это может быть описание идеи/спецификации/структуры/архитектуры
или отчёт о проделанной работе в репозитории.

Имя файла артефакта субагента начинается с `sa-`.

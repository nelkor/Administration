# TOOLS.md

## Репозиторий

Разработка приложения проходит внутри репозитория.
Расположен локально по пути `workspace/langley`.
Концепция продукта описана в `README.md` в корне репозитория.

## Артефакты

Текущие идеи, варианты и решения о разработке фиксируются в артефактах.

Расположены по пути `workspace/artifacts`. Содержит директории:
- `actual` — актуальные артефакты;
- `backlog` — временно отложенные;
- `archive` — исторические.

Основной агент перемещает файлы между директориями.

Субагент ориентируется только на актуальные. Из них он получает понимание
текущей ситуации и далее принимает решение (или совершает действие)
по своей конкретной задаче.

## Git

Внешних доступов нет, разработка и контроль версий производятся локально.

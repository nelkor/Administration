# Fix

Найди в репозитории продукта Pull Request с номером `target`.
Найди Issue этого Pull Request.

Прочитай Issue + комментарии и Pull Request + комментарии.

Разберись, что именно необходимо доработать.
Если получилось исправить проблемы — подготовь следующим шагом `review`.

Если постановка задачи, либо комментарии к Issue или Pull Request
тебя запутали и решить задачу не удалось, подробно опиши возникшие проблемы
в новом комментарии к Issue. Подготовить следующим шагом `reconsider`
с номером этого Issue в качестве `target`.

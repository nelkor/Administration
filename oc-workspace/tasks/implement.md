# Implement

> **ВАЖНО**
> Подготовка следующего шага — это только редактирование `state.json`.
> После подготовки следующего шага продолжать работу нельзя!

Найди в репозитории продукта Issue с номером `target`.
Прочитай Issue + комментарии.

Разберись, что именно необходимо сделать. Пиши код в ветке,
которая прикреплена к этому Issue (создай, если такой нет).

Если получилось решить задачу, создай Pull Request и подготовь
следующим шагом `review` с номером Pull Request в качестве `target`.

Если во время решения задачи выяснились факты, которые делают её решение
невозможным или нецелесообразным, подробно опиши возникшие проблемы
в новом комментарии к Issue. Подготовь следующим шагом `reconsider`
с номером этого Issue в качестве `target`.
